## Question 1: Domain of Rational Expressions

**Question:** Find the domain of the expression \(\dfrac{x-2}{x^2-16}\)

**Options:**
- A) \(\{x | x \neq 4\}\)
- B) \(\{x | x \neq -4\}\)
- C) \(\{x | x \neq \pm 4\}\) ✓
- D) \(\{x | x \neq 2\}\)

**Correct Answer:** C

**Explanation:** The denominator \(x^2 - 16 = (x-4)(x+4)\) equals zero when \(x = 4\) or \(x = -4\). Therefore, the domain excludes both values.

**Distractors:**
- A: Only considers positive root
- B: Only considers negative root
- D: Incorrectly uses numerator value

## Question 2: Simplifying Rational Expressions

**Question:** Simplify \(\dfrac{x^2-9}{x^2+6x+9}\)

**Options:**
- A) \(\dfrac{x-3}{x+3}\) ✓
- B) \(\dfrac{x+3}{x-3}\)
- C) \(\dfrac{x-9}{x+9}\)
- D) \(\dfrac{1}{x+3}\)

**Correct Answer:** A

**Explanation:** Factor numerator: \(x^2-9 = (x-3)(x+3)\). Factor denominator: \(x^2+6x+9 = (x+3)^2\). Cancel \((x+3)\) to get \(\dfrac{x-3}{x+3}\).

**Distractors:**
- B: Inverted numerator and denominator factors
- C: Did not factor, just split constant terms
- D: Incorrectly canceled \(x-3\) term

## Question 3: Multiplying Rational Expressions

**Question:** Simplify \(\dfrac{x^2-4}{x+3} \cdot \dfrac{x+3}{x+2}\)

**Options:**
- A) \(x+2\)
- B) \(x-2\) ✓
- C) \(\dfrac{x-2}{x+2}\)
- D) \(1\)

**Correct Answer:** B

**Explanation:** Factor \(x^2-4 = (x-2)(x+2)\). Multiply to get \(\dfrac{(x-2)(x+2)(x+3)}{(x+3)(x+2)}\). Cancel \((x+3)\) and \((x+2)\) to get \(x-2\).

**Distractors:**
- A: Used wrong sign
- C: Failed to cancel \((x+2)\)
- D: Over-simplified by canceling everything

## Question 6: Combining Radical Expressions

**Question:** Simplify \(3\sqrt{18} - 2\sqrt{8} + \sqrt{32}\)

**Options:**
- A) \(6\sqrt{2}\)
- B) \(9\sqrt{2}\) ✓
- C) \(13\sqrt{2}\)
- D) \(5\sqrt{58}\)

**Correct Answer:** B

**Explanation:** \(\sqrt{18} = 3\sqrt{2}\), \(\sqrt{8} = 2\sqrt{2}\), \(\sqrt{32} = 4\sqrt{2}\). So: \(3(3\sqrt{2}) - 2(2\sqrt{2}) + 4\sqrt{2} = 9\sqrt{2} - 4\sqrt{2} + 4\sqrt{2} = 9\sqrt{2}\).

**Distractors:**
- A: Arithmetic error in combining coefficients
- C: Added all coefficients without considering subtraction
- D: Did not simplify radicals before combining

## Question 7: Solving Rational Equations

**Question:** Solve \(\dfrac{4}{x} + \dfrac{3}{x-2} = 1\)

**Options:**
- A) \(x = 2\) or \(x = 4\)
- B) \(x = 1\) or \(x = 8\) ✓
- C) \(x = -1\) or \(x = 8\)
- D) \(x = 4\) only

**Correct Answer:** B

**Explanation:** LCD is \(x(x-2)\). Multiply through: \(4(x-2) + 3x = x(x-2)\). Expand: \(4x - 8 + 3x = x^2 - 2x\). Simplify: \(x^2 - 9x + 8 = 0\). Factor: \((x-1)(x-8) = 0\). Solutions: \(x = 1\) or \(x = 8\).

**Distractors:**
- A: Arithmetic error in solving quadratic
- C: Sign error
- D: Found only one solution

## Question 8: Identifying Functions

**Question:** Which of the following relations is **NOT** a function?

**Options:**
- A) \(\{(2,3), (3,4), (4,5), (5,6)\}\)
- B) \(\{(1,5), (1,7), (3,9)\}\) ✓
- C) \(\{(2,3), (3,3), (4,3), (5,3)\}\)
- D) \(f(x) = x^2 + 1\)

**Correct Answer:** B

**Explanation:** A relation is a function if each input has exactly one output. In option B, input 1 maps to both 5 and 7, violating the definition of a function.

**Distractors:**
- A: All inputs are unique
- C: Multiple inputs can map to same output (this is allowed)
- D: Algebraic functions always satisfy function definition

## Question 9: Intercepts of Linear Functions

**Question:** For the function \(f(x) = 2x - 8\), find the \(y\)-intercept as an ordered pair.

**Options:**
- A) \((4, 0)\)
- B) \((0, -8)\) ✓
- C) \((-4, 0)\)
- D) \((0, 8)\)

**Distractors:**

- C: Found x-intercept with wrong sign
- D: Sign error on constant term

## Question 10: Conjugates

**Question:** Find the conjugate of \(3 + \sqrt{11}\)

- A) \(3 - \sqrt{11}\) ✓
- B) \(-3 - \sqrt{11}\)
- C) \(-3 + \sqrt{11}\)

- C: Negated only the constant
- D: Reversed order (not standard conjugate form)

## Question 14: Dividing Rational Expressions

**Question:** Perform the division and simplify: \(\dfrac{x^2-16}{x^2+3x-10} \div \dfrac{x-4}{x+5}\)

- B) \(\dfrac{x+4}{x-2}\) ✓
- C) \(\dfrac{x+4}{x+5}\)

**Explanation:** Multiply by reciprocal: \(\dfrac{x^2-16}{x^2+3x-10} \cdot \dfrac{x+5}{x-4}\). Factor: \(\dfrac{(x-4)(x+4)}{(x+5)(x-2)} \cdot \dfrac{x+5}{x-4}\). Cancel \((x-4)\) and \((x+5)\): \(\dfrac{x+4}{x-2}\).

**Distractors:**
- A: Sign error
- C: Failed to cancel all common factors
- D: Over-simplified denominator

## Question 15: Adding Rational Expressions

**Question:** Perform the addition and simplify: \(\dfrac{2}{x-3} + \dfrac{5}{x+2}\)

**Options:**
- A) \(\dfrac{7}{x-1}\)
- B) \(\dfrac{7x-11}{(x-3)(x+2)}\) ✓
- C) \(\dfrac{7x+11}{(x-3)(x+2)}\)
- D) \(\dfrac{7}{2x-1}\)

**Correct Answer:** B

**Explanation:** LCD is \((x-3)(x+2)\). Rewrite: \(\dfrac{2(x+2)}{(x-3)(x+2)} + \dfrac{5(x-3)}{(x-3)(x+2)} = \dfrac{2x+4+5x-15}{(x-3)(x+2)} = \dfrac{7x-11}{(x-3)(x+2)}\).

**Distractors:**
- A: Incorrectly simplified denominator and numerator
- C: Sign error in combining constants
- D: Completely wrong LCD and simplification

## Question 17: Solving Rational Equations with Verification

**Question:** Solve the equation: \(\dfrac{4}{x-3} - \dfrac{2}{x} = 1\)

**Options:**
- A) \(x = 3\) or \(x = 2\)
- B) \(x = 6\) or \(x = -1\) ✓
- C) \(x = 6\) only
- D) \(x = -6\) or \(x = 1\)

**Correct Answer:** B

**Explanation:** LCD is \(x(x-3)\). Multiply through: \(4x - 2(x-3) = x(x-3)\). Expand: \(4x - 2x + 6 = x^2 - 3x\). Rearrange: \(x^2 - 5x - 6 = 0\). Factor: \((x-6)(x+1) = 0\). Solutions: \(x = 6\) or \(x = -1\).

**Distractors:**
- A: Used restricted values instead of solving
- C: Found only one solution
- D: Sign error in factoring

## Question 18: Rationalizing with Conjugates

**Question:** Rationalize the denominator and simplify: \(\dfrac{8}{4-\sqrt{3}}\)

**Options:**
- A) \(\dfrac{32 - 8\sqrt{3}}{13}\)
- B) \(\dfrac{32 + 8\sqrt{3}}{13}\) ✓
- C) \(\dfrac{8 + 2\sqrt{3}}{13}\)
- D) \(\dfrac{4 + \sqrt{3}}{2}\)

**Correct Answer:** B

**Explanation:** Multiply by conjugate \(\dfrac{4+\sqrt{3}}{4+\sqrt{3}}\): \(\dfrac{8(4+\sqrt{3})}{(4-\sqrt{3})(4+\sqrt{3})} = \dfrac{32+8\sqrt{3}}{16-3} = \dfrac{32+8\sqrt{3}}{13}\).

**Distractors:**
- A: Sign error in numerator
- C: Arithmetic error in multiplying numerator
- D: Incorrect conjugate multiplication

## Question 20: Function Evaluation (Part b)

**Question:** Given the function \(f(x) = x^2 - 3x + 5\), find \(f(a+2)\).

**Options:**
- A) \(a^2 + a + 3\) ✓
- B) \(a^2 + 4a + 7\)
- C) \(a^2 - 3a + 7\)
- D) \(a^2 + a - 1\)

**Distractors:**
- B: Did not distribute -3 correctly
- C: Did not expand \((a+2)^2\) correctly
- D: Arithmetic error in combining constants
